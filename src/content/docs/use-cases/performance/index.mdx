Accelerate your website or application with global caching, image optimization, and smart routing. Cloudflare Cache serves content from edge locations globally. Images and Polish optimize and compress images on-the-fly. Argo Smart Routing avoids congested network paths. Load Balancing distributes traffic with health checks and failover. Web Analytics tracks Core Web Vitals from real user sessions.

<DirectoryListing />

## Prerequisites

- A [Cloudflare account](https://dash.cloudflare.com/sign-up).
- A domain [added to Cloudflare](/fundamentals/manage-domains/add-site/) with DNS records proxied through Cloudflare's network. Caching, image optimization, speed optimizations, Argo Smart Routing, and Load Balancing all operate at the edge and require traffic to pass through Cloudflare.
- Web Analytics can be added to any site without a proxied domain by [adding the JavaScript snippet](/web-analytics/get-started/) to your pages directly.
